# cursortrial

A trial repository for experimenting with [Cursor](https://cursor.com) and GitHub workflows.

## About

This repo is a sandbox for testing Cursor features such as AI-assisted coding, repository setup, and version control.

## Getting started

1. Clone the repository:
   ```bash
   git clone https://github.com/bagusawann/cursortrial.git
   cd cursortrial
   ```
2. Open the folder in Cursor.
3. Start building or experimenting.
